# Http server

## Start
   java -jar httpserver.jar -help  

Usage: [Options]  
Options:  
-help <arg>     Display help information.  
-p <arg>        Port. 8080 will be used if not provided.  
-h <arg>        Host name. localhost will be used if not provided  
-d <arg>        Document Base(absolute or relative)  
                Current folder will be used if not provided
